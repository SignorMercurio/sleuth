# MITRE ATT&CK 框架参考

本文档提供常见 ATT&CK 战术和技术的快速参考，用于应急响应报告中的攻击行为映射。

## 使用说明

- 战术（Tactics）：攻击者的战术目标（"为什么"）
- 技术（Techniques）：实现战术目标的具体方法（"如何做"）
- 编号格式：T[数字] 或 T[数字].[子编号]

---

## TA0001 - 初始访问 (Initial Access)

攻击者试图进入目标网络。

### T1190 - 利用面向公众的应用程序
利用 Web 应用、数据库、服务等的漏洞获得初始访问权限。
- 常见场景：SQL 注入、远程代码执行 (RCE)、文件上传漏洞
- 检测：Web 访问日志中的异常请求、漏洞扫描特征

### T1133 - 外部远程服务
通过 VPN、Citrix、SSH 等外部远程服务访问。
- 常见场景：SSH 暴力破解、VPN 凭据泄露
- 检测：异常的远程登录、多次失败登录尝试

### T1566 - 钓鱼攻击
- **T1566.001** - 附件钓鱼：通过恶意附件投递恶意软件
- **T1566.002** - 链接钓鱼：通过恶意链接诱导用户访问
- 检测：邮件日志、用户行为分析

### T1078 - 有效账户
使用合法的账户凭据进行访问。
- **T1078.001** - 默认账户
- **T1078.003** - 本地账户
- 检测：异常时间/地点的登录、权限使用异常

## TA0002 - 执行 (Execution)

攻击者试图运行恶意代码。

### T1059 - 命令和脚本解释器
- **T1059.001** - PowerShell
- **T1059.003** - Windows 命令 Shell (cmd.exe)
- **T1059.004** - Unix Shell (bash, sh, zsh)
- **T1059.006** - Python
- **T1059.007** - JavaScript
- 检测：进程执行日志、命令行参数分析

### T1053 - 计划任务/作业
- **T1053.003** - Cron (Linux/macOS)
- **T1053.005** - 计划任务 (Windows)
- 检测：定时任务配置变更、可疑的定时任务条目

### T1203 - 利用客户端执行
利用客户端软件的漏洞（浏览器、Office、PDF 阅读器等）执行代码。
- 检测：崩溃日志、异常的子进程

### T1204 - 用户执行
诱导用户执行恶意内容。
- **T1204.001** - 恶意链接
- **T1204.002** - 恶意文件
- 检测：用户行为分析、下载记录

### T1136 - 创建账户

## TA0004 - 权限提升 (Privilege Escalation)

攻击者试图获取更高权限。

### T1068 - 利用漏洞提权
利用操作系统或软件漏洞提升权限。
- 常见场景：内核漏洞利用、本地提权漏洞
- 检测：异常的权限提升、内核模块加载

### T1548 - 滥用提升控制机制
- **T1548.001** - Setuid 和 Setgid
- **T1548.003** - Sudo 和 Sudo Caching
- 检测：sudo 日志、SUID 文件变更

### T1053 - 计划任务/作业
利用计划任务提权。
- 检测：以高权限运行的定时任务

### T1543 - 创建或修改系统进程
创建以高权限运行的服务。
- 检测：服务配置变更

### T1078 - 有效账户
使用或窃取高权限账户。
- 检测：特权账户的异常使用

- **T1562.003** - 削弱或禁用云日志
- **T1562.004** - 禁用或修改系统防火墙
- 检测：安全工具状态监控、防火墙规则变更

### T1036 - 伪装

- **T1036.005** - 匹配合法名称或位置

### T1027 - 混淆文件或信息

### T1222 - 文件和目录权限修改

## TA0007 - 发现 (Discovery)

攻击者试图了解目标环境。

### T1082 - 系统信息发现
收集系统版本、配置、补丁信息等。
- 常见命令：uname, cat /etc/os-release, systeminfo
- 检测：信息收集命令执行

### T1083 - 文件和目录发现
枚举文件系统以查找感兴趣的文件。
- 常见命令：find, ls, tree
- 检测：大范围文件系统遍历

### T1046 - 网络服务扫描
扫描网络以发现可用的服务和主机。
- 检测：端口扫描流量、nmap 等工具特征

### T1016 - 系统网络配置发现
- 常见命令：ifconfig, ip addr, netstat, route
- 检测：网络配置查询

### T1033 - 系统所有者/用户发现
识别当前用户和其他用户账户。
- 常见命令：whoami, who, w, id
- 检测：用户枚举行为

### T1069 - 权限组发现
- **T1069.001** - 本地组
- **T1069.002** - 域组
- 常见命令：groups, id, cat /etc/group
- 检测：组成员枚举

### T1057 - 进程发现
列出正在运行的进程。
- 常见命令：ps, top, tasklist
- 检测：频繁的进程枚举

### T1018 - 远程系统发现
识别网络中的其他系统。
- 检测：网络扫描、ARP 请求

## 使用提示

1. **多个战术映射**：一个攻击行为可能同时映射到多个战术（如定时任务可以是执行、持久化、权限提升）
2. **具体化描述**：在报告中应描述具体的攻击手法，而不仅仅列出编号
3. **证据支撑**：每个 ATT&CK 映射都应有对应的溯源证据
4. **子技术优先**：当适用时，使用更具体的子技术编号（如 T1059.004 而不是 T1059）
